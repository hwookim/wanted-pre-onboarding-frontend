# 원티드 프리온보딩 프론트엔드 - 선발 과제

## 프로젝트 실행 방법

```shell
$ git clone https://github.com/hwookim/wanted-pre-onboarding-frontend.git
$ cd wanted-pre-onboarding-frontend
$ npm install
$ npm start
```
